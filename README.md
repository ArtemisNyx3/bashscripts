## File Management

## Usage

1. Clone the repo:

```bash
git clone https://github.com/yourusername/my-scripts.git
cd bashscripts
```

2. Make scripts execurtable (if needed):

```bash
chmod +x some_script.sh
```

3. Run the script

```bash
./some_script.sh
```
